# DynamoDb-Item-Size

DynamoDb-Item-Size is a Nodejs library for calculate the byte size of an object.

## Installation


```bash
npm install dynamodb-item-size
```

## Usage

```javascript
const { calculateItemSize } = require('dynamodb-item-size');
```
